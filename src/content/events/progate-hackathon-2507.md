## 技術スタック
- **Languages**: TypeScript, Python

## 概要
Event link: https://progate.connpass.com/event/357899/  
Progateハッカソンに初参加した  
ハッカソンのテーマが「夏の風物詩」で、小学生の夏休みの自由研究をテーマに、AIサポートのWebアプリ開発を行った  
AWS環境が提供されることもあり、Amazon Bedrockでプロンプトを使用した  
チームとしてもほぼ初参加メンバーで構成され、AIコーディング環境も準備できていなかったため、デプロイまでは至らなかった  

- Topa'z(発表資料): https://topaz.dev/projects/3b6b75ea5cf110207f9c
- GitHub Repository: https://github.com/karen-kac/summer

## 経験した開発
バックエンドのアーキテクチャ設計と、フロントエンドのコードレビューを主に行った  
特に、チームメンバーがAIエージェントフル活用でフロントエンド実装を行ってくれたため、フロントエンドとバックエンドと連携しやすくなるようなコードレビュー・アーキテクチャ設計を行った
