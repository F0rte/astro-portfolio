## 技術スタック
- **Languages**: TypeScript, Node.js
- **DevOps/CI**: Docker, Docker Compose, GitHub Actions, ESLint, Prettier

## 概要
event link: https://progate.connpass.com/event/354577/  
Gitの内部構造を理解して、自作するというテーマで実装タスクに取り組む  
AsanaやDesign Docを活用し、設計からタスク分割、実装までを通し、チームでの開発を経験できた  

GitHub Repository: https://github.com/Progate/git-intern-2025-summer-term2

## 経験した開発
設計段階で、内部構造を理解しながらアーキテクチャ設計にこだわった  
TypeScriptに慣れていなかったことと、実装自体はスピードを重視するため、Vibe Codingを行った  
[ARCHITECTURE.md](https://github.com/Progate/git-intern-2025-summer-term2/blob/main/ARCHITECTURE.md)でGeminiで設計書を作成、コーディングにはGitHub Copilotを使用した  
PRでセルフチェックを行うなど、コードレビューを人力で行った  
また、デバッグを容易にするため、Dockerを用いた最終確認テストを実装し、最終日の短い時間でバグ解消に取り組むことができた
